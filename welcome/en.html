Frequently asked questions

Everything you need to know to get the most out of Notebook LM Importer.

Navigate to any YouTube video and click the Notebook LM Importer icon in your Chrome toolbar. Select the notebook you want to add it to (or create a new one), then click "Add to Notebook". Alternatively, the extension adds import buttons directly on YouTube pages — look for the orange "Add" button next to the video title.

Yes! Open a YouTube playlist page or a channel page and click the extension icon. You'll see options to "Add Playlist" or "Add Channel Videos". The extension extracts all visible video URLs and imports them at once. For large channels, the first ~50 videos visible in the DOM are imported.

Click the extension icon and press "Bulk Import". A full-page interface opens where you can paste multiple URLs (one per line). Select a target notebook and click "Import Links". All sources are added simultaneously. You can also switch to the "From Tabs" tab to import from currently open browser tabs.

Open the extension popup and click "Bulk Import", then switch to the "From Tabs" tab. You'll see all your currently open tabs. Check the ones you want to import, select a notebook, and click "Import Selected". This is perfect when you've been researching a topic across multiple pages.

The Page Content feature extracts the full text of the current web page and imports it as a text source into Notebook LM. This is especially useful for paywalled articles, restricted pages, or PDFs that Notebook LM can't fetch directly on its own — as long as you have access to the page in your browser, the extension can import it.

Click the extension icon and open Settings (gear icon at the bottom). Under "Account", select the Google account you want to use. The extension lists all Google accounts currently signed in to your browser. After switching, notebooks from the selected account will be loaded automatically.

Enable the "Bulk Delete" feature in extension Settings. Once enabled, a delete button appears in Notebook LM whenever you select multiple sources. You can also use the "Manage Sources" tab in the Bulk Import interface to load all sources in a notebook and delete them in bulk directly from there.

Yes. The extension communicates directly with the Notebook LM API using your active Google session. Make sure you are logged in to notebooklm.google.com before using the extension. No passwords or tokens are stored — your existing browser session handles authentication automatically.
