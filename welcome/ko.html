FAQ

자주 묻는 질문

Notebook LM Importer를 최대한 활용하기 위해 알아야 할 모든 것.

YouTube 동영상으로 이동하여 Chrome 도구 모음의 Notebook LM Importer 아이콘을 클릭하세요. 추가할 노트북을 선택하고 "노트북에 추가"를 클릭하세요. YouTube 페이지의 비디오 타이틀 옆에 주황색 "추가" 버튼도 표시됩니다.

네! 재생목록 또는 채널 페이지를 열고 확장 프로그램 아이콘을 클릭하세요. "재생목록 추가" 또는 "채널 동영상 추가" 옵션이 표시됩니다. 대형 채널은 처음 ~50개 동영상이 가져와집니다.

확장 프로그램 아이콘을 클릭하고 "대량 가져오기"를 선택하세요. 여러 URL(한 줄에 하나씩)을 붙여넣고 노트북을 선택한 후 "링크 가져오기"를 클릭하세요.

확장 프로그램 팝업을 열고 "대량 가져오기"를 클릭한 다음 "탭에서" 탭으로 전환하세요. 가져올 탭을 선택하고 "선택 항목 가져오기"를 클릭하세요.

현재 웹 페이지의 전체 텍스트를 추출하여 Notebook LM에 텍스트 소스로 가져옵니다. 유료 기사나 제한된 페이지에 특히 편리합니다.

확장 프로그램 아이콘을 클릭하고 설정(기어 아이콘)을 여세요. "계정" 아래에서 원하는 Google 계정을 선택하세요.

확장 프로그램 설정에서 "대량 삭제"를 활성화하세요. 여러 소스를 선택하면 삭제 버튼이 표시됩니다.

네. 확장 프로그램은 활성 Google 세션을 사용하여 Notebook LM API와 통신합니다. 사용 전에 notebooklm.google.com에 로그인되어 있는지 확인하세요.
